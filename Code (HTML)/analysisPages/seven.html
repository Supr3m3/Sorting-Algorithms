Сортування пірамідою

Пірамідальна сортування (вона ж сортування купою) - класичний алгоритм який, мабуть, повинен знати будь-який програміст. Стара добра «пірамідка» примітна тим, що в незалежності від набору даних у неї одна і та ж складність за часом (причому, дуже пристойна) - O (n log n). Кращих і вироджених випадків для неї немає. Масив можна впорядкувати, якщо на його основі будувати і перебудовувати Сортувальне дерево, відоме як бінарна купа або просто піраміда. Що є Сортувальне дерево? Це дерево, у якого будь-який батько більше (або менше, дивлячись в який бік воно вбиральня) ніж його нащадки. Як зі звичайного дерева зробити Сортувальне дерево? Дуже просто - потрібно рухатися від нащадків вгору до батьків і якщо нащадок більше батьків, то міняти місцями. Якщо такий обмін відбувся, неохайного на один рівень з батьків потрібно порівняти з нащадками нижче - може і там теж буде привід для обміну. Реалізація:

C++

for(j = 0; j < n - 1; j++)
for(i = 0; i < n - 1; i++)
if (arr[i] > arr[i + 1])
tmp = arr[i];
arr[i] = arr[i + 1];
arr[i + 1] = tmp;
