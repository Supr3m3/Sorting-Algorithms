Сортування плавне

Був винайдений відомим Е. В. Дейкстрой в 1981 році і є розвитком ідеї Heapsort - сортування за допомогою двійковій купи (ну або піраміди, кому як більше подобається). Для початку розповім, як працює це сортування: Спочатку слід побудувати купу, додаючи в неї по одному елементу (складність O (log n) на кожен, всього O (n log n)) Витягувати максимальний елемент, який лежить в корені побудованої купи, до тих пір, поки вона не стане порожня. Так як при видаленні кореня купа розпадається на дві незалежні, потрібно зібрати їх в одну за O (log n). В сумі для n елементів - також O (n log n) Існує оптимізація, що дозволяє виконати перший крок зі складністю O (n), але другий крок все одно матиме складність O (n log n). Алгоритму потрібно O (1) додаткової пам'яті, купа зберігається прямо в початковому масиві з даними. 

                            

                               
C++

                                for(j = 0; j < n - 1; j++){
                                  for(i = 0; i < n - 1; i++){
                                    if (arr[i] > arr[i + 1]) {
                                    tmp = arr[i];
                                    arr[i] = arr[i + 1];
                                    arr[i + 1] = tmp;
                                    }
                                  }
                                }
