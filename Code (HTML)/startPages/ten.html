Сортування рандомне (бого)

Претендент на звання найповільнішого сортування. Масив перемішується до тих пір поки не відсортуєтся. У найгіршому випадку, час сортування: ∞; В середньому, час сортування: O(n x n!). Як відомо, для масиву містить n елементів існує n! перестановок. Перемішаємо масив. В результаті цієї дії, Всесвіт розщепиться на n! паралельних світів, в кожному з яких своє розташування елементів в шуканому масиві. Переміщаємося з Всесвіту у Всесвіт і в кожної перевіряємо масив. Якщо масив не впорядкований, то знищуємо Всесвіт. У підсумку залишаться тільки ті Всесвіти, в яких цей масив відсортований. 

                            

                               
C++

                                for(j = 0; j < n - 1; j++){
                                  for(i = 0; i < n - 1; i++){
                                    if (arr[i] > arr[i + 1]) {
                                    tmp = arr[i];
                                    arr[i] = arr[i + 1];
                                    arr[i + 1] = tmp;
                                    }
                                  }
                                }
