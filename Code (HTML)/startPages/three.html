Сортування швидке

Алгоритм Quick Sort є одним з найшвидших алгоритмів сортування. Його середній час виконання - O (n log n). Quick Sort реалізується за допомогою рекурсії. У цій статті представлений демонстраційний приклад алгоритму швидкого сортування на Java. Суть алгоритму полягає в наступному. Береться певний елемент масиву (бажано Рандома, щоб алгоритм виконувався приблизно однакові проміжки часу для будь-яких наборів даних). Елементи, які знаходяться зліва і більше за значенням ніж початковий, міняються місцями з елементами, які знаходяться праворуч і менше початкового. Аналогічна операція виконується для наборів даних зліва і праворуч від початкового. І так далі по рекурсії. 
                            
                               /* low  --> Starting index,  high  --> Ending index */
                               void quickSort(arr[], low, high)
                                {
                                    if (low < high)
                                    {
                                        /* pi is partitioning index, arr[p] is now
                                           at right place */
                                        pi = partition(arr, low, high);

                                        quickSort(arr, low, pi - 1);  // Before pi
                                        quickSort(arr, pi + 1, high); // After pi
                                    }
                                }
